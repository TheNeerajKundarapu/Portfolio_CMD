# Portfolio OS - Command Line Interface Portfolio

A premium, modern command-line interface (CLI) portfolio website built with React and Vite. featuring a "Cyber-Glow" aesthetic, glassmorphism, and realistic CRT effects.

![Premium UI Redesign](https://github.com/TheNeerajKundarapu/Portfolio_CMD/raw/main/src/assets/preview.png) *(Note: Add a screenshot to assets for a better experience)*

## ✨ Features

- **Cyber-Glow Aesthetic**: A sophisticated color palette with deep slate, neon cyan, and purple accents.
- **Glassmorphism**: Translucent terminal window with frosted-glass effects and subtle border glows.
- **CRT Effects**: Realistic scanlines, flicker, and phosphor-like glow for a retro-future feel.
- **Interactive Terminal**: Fully functional CLI with command processing.
- **Live Uptime**: Dynamic uptime counter in the terminal header.
- **Responsive Design**: Designed to scale and look great on various screen sizes.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16.0.0 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/TheNeerajKundarapu/Portfolio_CMD.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Portfolio_CMD
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Built With

- **React**: Frontend library for building the UI.
- **Vite**: Ultra-fast build tool and development server.
- **Vanilla CSS**: Custom styling for maximum control and performance.
- **JetBrains Mono**: Premium typography for that developer feel.
